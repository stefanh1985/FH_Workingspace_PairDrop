# 📂 Was ist das für ein Programm?

![Pairdrop-Logo](https://avatars.githubusercontent.com/u/52242352?s=48&v=4)

PairDrop ist eine Technologie, die es zwei Geräten ermöglicht, Daten direkt und einfach miteinander zu teilen, indem sie physisch nah beieinander sind. Stellen Sie sich vor, Sie möchten ein Foto von Ihrem Smartphone auf das eines Freundes übertragen. Mit PairDrop könnten Sie dies tun, indem Sie die Geräte nebeneinander halten, ohne eine komplizierte Einrichtung durchführen zu müssen. Die Technologie nutzt verschiedene drahtlose Kommunikationsformen, wie Bluetooth oder WLAN, um eine sichere Verbindung zwischen den Geräten herzustellen. Einmal verbunden, können Dateien, Fotos, Kontakte und sogar manche Einstellungen schnell und sicher übertragen werden. Dies ist besonders nützlich in Situationen, in denen keine Internetverbindung verfügbar ist oder wenn die Datensicherheit eine große Rolle spielt, da die Daten direkt von einem Gerät zum anderen übertragen werden, ohne über externe Server zu gehen. PairDrop ist so konzipiert, dass es intuitiv und benutzerfreundlich ist, sodass selbst Personen ohne technische Vorkenntnisse die Funktion leicht nutzen können. Um PairDrop zu verwenden, müssen beide Geräte diese Funktion unterstützen und aktiviert haben. Die Technologie dahinter ist darauf ausgerichtet, die Interaktion zwischen Geräten so nahtlos und effizient wie möglich zu gestalten. PairDrop eröffnet eine neue Dimension der Datenübertragung, die sowohl praktisch als auch sicher ist, und macht den Austausch von Informationen zwischen Geräten einfacher denn je.

**Zusammenfassung**

 * Gemeinsame Nutzung von Dateien in Ihrem lokalen Netzwerk:  Senden Sie Bilder, Dokumente oder Texte über eine Peer-to-Peer-Verbindung an Geräte im selben lokalen Netzwerk.
*  Internet-Übertragungen: Treten Sie temporären öffentlichen Räumen bei, um Dateien einfach über das Internet mit einer sicheren Verbindung zu übertragen!
*     Web-Anwendung: Da PairDrop webbasiert ist, läuft es auf allen Geräten.

# 🧑‍🏫 Wozu wird es in der Lehre genutzt?
PairDrop ist eine einfache, webbasierte Plattform, die den Austausch von Dateien zwischen Geräten im selben Netzwerk ermöglicht, ideal für die Nutzung an einer Fachhochschule. Dozierende können damit schnell Dokumente, Präsentationen oder Lehrmaterialien direkt an die Studierenden senden, ohne E-Mails oder USB-Sticks nutzen zu müssen. Studierende können ihrerseits Hausarbeiten oder Projekte einfach an ihre Lehrkräfte oder untereinander übermitteln. Die Nutzung erfordert lediglich eine Internetverbindung und den Zugriff auf die PairDrop-Webseite. Es ist keine Installation oder Registrierung nötig, was Zeit spart und Datenschutzbedenken minimiert. PairDrop unterstützt verschiedene Dateitypen, von Textdokumenten bis hin zu Videos, was es zu einem vielseitigen Werkzeug macht. Durch die Einfachheit der Plattform können auch Nutzer*innen ohne technische Vorkenntnisse problemlos Dateien austauschen. Da die Übertragung direkt von Gerät zu Gerät erfolgt, ist der Prozess schnell und effizient. Dozierende können auch Feedback zu eingereichten Aufgaben geben, indem sie annotierte Dateien zurücksenden. PairDrop fördert somit eine effiziente und interaktive Lernumgebung an der Fachhochschule.

## 🧑‍🦱Nutzer*innenperspektive
Einfach auf den Geräten die folgende Webseite aufrufen:

https://pairdrop.net

Sind die Geräte im selben Netzwerk werden sie sofort erkannt und sind gegenseitig sichtbar. Die Namen sind anonym, können aber von den Teilnehmenden individualisiert werden. Die Übertragung erfolgt dann wie bei Airdrop. Zieldevice und Datei auswählen und los gehts. Auf dem Empfangsgerät muss der Transfer zuerst noch bestätigt werden.

Sind die Geräte in unterschiedlichen Netzwerken (z. B. Wlan und Handy-Netz) kann ein öffentlicher Raum erstellt werden und per Link oder QR-Code geteilt werden. Die Sichtbarkeit und Übertragung geht dann genauso, wie im selben Netzwerk über die Bühne.
